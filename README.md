# MetodosNumericos

Este es un proyecto que pretender crear un programa escrito en Matlab que reuna
múltiples métodos numéricos existentes. Principalmente reune los métodos
numéricos más populares.

## Comenzando a usarlo
### Prereqisitos

Como prerequisito para poder ejecutar el proyecto necesita tener instalado una
versión de Matlab R2017a o superior.

No garantizamos que funcione en versiones anteriores de Matlab, pero puede
intentar correrlo en una versión anterior.

### Iniciar el programa

Para iniciar el programa de forma correcta debe abrir Matlab y ubicar el
directorio del proyecto como el directorio actual de trabajo de Matlab.
Luego debe ejecutar el script Software_Metodos.m y empezar a escoger el método
con el que desea resolver un problema e ingresar los parámetros dentro de los
cuadros de texto.
